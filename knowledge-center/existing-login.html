Use Existing Login.gov Account

Follow these steps if you have an existing Login.gov account to get started with the TANF Data Portal (TDP).

  1. Sign into the TANF Data Portal

    Begin here: Visit the TANF Data Portal and select "Sign in with LOGIN.GOV for grantees"

  2. Sign-In

    • Enter Email Address
    • Enter Password
    • Click "Sign-in"

  3. Request Access

    Request access to the TANF Data Portal.

    • Enter First Name
    • Enter Last Name
    • Select your Associated State, Tribe or Territory

  4. Request Submitted

    Your request for access is being reviewed by an OFA admin. You will receive an email when access has been granted. Please allow up to 1 business day for a response.

  5. Await for Access Next Steps

    Within 1 business day, your request for access will be reviewed. Please check your inbox for an email from tdppilot@goraft.tech on next steps.

  6. Approved for the TANF Data Portal

    You are approved to begin working in the TANF Data Portal.
